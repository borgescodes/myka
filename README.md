# Sudoku da minha diva

Um Sudoku estatico, fofo e romantico feito com HTML, CSS e JavaScript puro. O projeto funciona abrindo o `index.html` no navegador e tambem fica pronto para publicar no GitHub Pages.

## Arquivos

- `index.html`: estrutura da pagina e do jogo.
- `style.css`: identidade visual rosa, suave e kawaii.
- `script.js`: geracao do Sudoku, validacao, timer, dicas, mensagens e salvamento local.
- `assets/`: pasta opcional para imagens personalizadas.

## Assets opcionais

O jogo nao depende de imagens externas. Se quiser personalizar depois, adicione estes arquivos na pasta `assets`:

- `assets/logo.png`
- `assets/cat.png`
- `assets/bow.png`
- `assets/heart.png`
- `assets/background-pattern.png`

Caso algum arquivo nao exista, o layout usa automaticamente elementos fofos feitos com HTML/CSS.

## Como publicar no GitHub Pages

1. Crie um repositorio no GitHub.
2. Envie os arquivos `index.html`, `style.css`, `script.js`, `README.md` e a pasta `assets`.
3. No GitHub, entre em `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/root`.
6. Acesse o link gerado pelo GitHub Pages.

## Rodando localmente

Basta abrir o arquivo `index.html` no navegador. Nao precisa instalar nada, rodar servidor, banco de dados ou backend.

## Observacao sobre inspiracao visual

A estetica foi criada para lembrar um universo delicado e kawaii com gatinha generica, lacos, coracoes e tons de rosa/lilas. O projeto nao usa imagens oficiais nem assets protegidos.
